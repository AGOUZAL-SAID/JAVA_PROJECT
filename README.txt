
🏭 Robotism - Factory Simulation  

📌 Description  
Robotism est un projet de simulation graphique représentant une usine où des robots se déplacent entre différentes zones en trouvant le chemin le plus court à l’aide de l’algorithme de Dijkstra. Ce projet est développé en Java dans le cadre de la première année d'ingénierie à Télécom Paris.  

🚀 Fonctionnalités  
- Simulation graphique d’une usine avec plusieurs zones  
- Déplacement intelligent des robots en utilisant Dijkstra  
- Interface interactive pour observer la simulation  

🛠 Installation et exécution  
1. Cloner le dépôt
   ```
   git clone git@github.com:AGOUZAL-SAID/JAVA_PROJECT.git
   cd robotism
   ```
2. Compiler et exécuter la simulation  
   - Assurez-vous d’avoir Java installé.  
   - Lancer l’application depuis le fichier principal :  
     ```
     cd src/robotism
     java SimulatorApplication
     ```
